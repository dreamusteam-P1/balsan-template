# 고객사 요구사항 및 향후 관리 계획

## 📌 고객사 직접 관리 기능 (추후 구현)
고객사가 홈페이지 완성 후 스스로 내용을 수정하고 파일을 관리할 수 있도록 하는 기능들을 나중에 검토하고 구현할 예정입니다.

### 1. 주요 관리 대상
- **사용설명서 (Manuals):** 텍스트 수정 및 최신 버전 업데이트
- **카탈로그 (Catalogs):** PDF 파일 업로드 및 다운로드 링크 관리
- **이미지 및 리소스:** 저장 위치(스토리지) 지정 및 손쉬운 교체

### 2. 제안된 구현 방식 (최종 단계에서 결정)
- **CMS 연동 (Sanity 등):** 가장 직관적이고 쉬운 관리 방식
- **전용 관리자 대시보드:** 맞춤형 UI 제공
- **클라우드 스토리지 연동:** 구글 드라이브나 AWS S3 등을 통한 파일 자동 업데이트

---
> **Note:** 현재는 디자인 및 기능 구현 단계입니다. 고객사의 최종 승인이 나면, 위 방식 중 최적의 솔루션을 선택하여 적용합니다.
